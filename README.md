# Smart AI Timetable Generator

Custom Input Based Timetable Generator using Constraint Scheduling

Features:
- 5 Days | 6 Periods
- Exactly 3 Classes per Subject per Week
- Custom Teachers
- Custom Rooms
- Custom Subjects
- No Teacher Clash
- No Room Clash
- Free Periods Displayed as '-'
- Subject not repeated twice in same day
